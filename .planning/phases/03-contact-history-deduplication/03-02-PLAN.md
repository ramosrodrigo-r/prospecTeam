Testes a incluir (5 testes):

2. `'retorna todos os prospects quando historico esta vazio'` — loadHistory() (sem arquivo), prospects = [{placeId: 'A', name: 'A'}, {placeId: 'B', name: 'B'}], assert.equal(dedupProspects(prospects).length, 2)

4. `'retorna vazio quando todos sao duplicatas'` — writeFileSync com `{"X": {"sentAt": "..."}, "Y": {"sentAt": "..."}}`, loadHistory(), prospects = [{placeId: 'X'}, {placeId: 'Y'}], assert.equal(dedupProspects(prospects).length, 0)
